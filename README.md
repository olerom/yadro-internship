Тестовое задание для стажировки

## Quick start
Pull image

``docker pull olerom/yadro-internship``

Run container

``docker run -it <IMAGE ID>``

Examples

```
cli_net help
cli_net list --server 127.0.0.1 --port 8080
cli_net show eth0 --server 127.0.0.1 --port 8080
cli_net quit
```
